# movie.search
Movie search using an existing online movie repository for general information about movies and series: https://www.themoviedb.org/documentation/api, Using mediator pattern with 3-layer for clean architecture 

NuGet packages
-Mediatr
-EasyCaching
-TMDbLib
-Automapper

Missing features
-Doesn't include TvServies
-Doesn't include Trailers
-Validations fields 
-Error Handling 
-Retry policy 
-API Authentication
